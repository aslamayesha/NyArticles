## Code Formatting & Linting


### 1.  Format Code

```bash
npm run format

``` 
 
### 2. Lint Code
To check your code for any linting errors and automatically fix them, run:

```bash
npm run lint

```

### 3. Fix Linting Issues
If you want to automatically fix issues detected by the linter, use:

```bash
npm run lint:fix

```
